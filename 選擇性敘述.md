# 1. if

```
if(判斷條件true)
{
  執行此處
}
```
```c
int num = 5;
/*如果num>0, 則執行以下程式*/
if(num>0)
  printf("你輸入的值>0\n"); /*一個敘述，不用{}*/
    
printf("程式結束");
```
# 2. if-else

```
if(判斷條件true)
{
  執行此處
}
else
{
  判斷條件false, 執行此處
}
```
```c
int num = -1;
    
if(num>0)
    printf("num的值>0\n");
else
    printf("num的值<=0\n"); /*當num<=0時，執行這裡*/        
printf("程式結束");
```
