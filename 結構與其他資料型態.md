# Structure(結構)
1.功能: 將一群型態不同的資料組合在一起。  
2.結構的定義:
```
struct 結構名稱           /*定義結構*/
{
  資料型態 成員名稱1;      /*宣告結構成員*/
  資料型態 成員名稱2;
        ...
  資料型態 成員名稱n;
};
```
3.宣告結構變數:
```
struct 結構名稱 變數1, 變數2, ..., 變數n;         
```
4.合併後:
```
struct 結構名稱 
{
  資料型態 成員名稱1;      
  資料型態 成員名稱2;
        ...
  資料型態 成員名稱n;
} 變數1, 變數2, ..., 變數n;
```
